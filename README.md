# S. Karthikeyan — Resume Site

A single-page resume site with a "ledger" design theme (accounts/MIS aspirant)
and a looping background video in the header, matched to the site's colour
palette: Crimson Depth `#710014`, Warm Sand `#B38F6F`, Soft Pearl `#F2F1ED`,
Obsidian Black `#161616`.

## Files
- `index.html` — the whole site (HTML + CSS in one file)
- `assets/bg-video.mp4` — compressed, looping header background video
- `assets/bg-poster.jpg` — poster frame shown before the video loads

## Run locally
Just open `index.html` in a browser — no build step needed.

## Deploy with GitHub Pages
1. Push this folder's contents to a GitHub repo (root of the repo, or a `/docs` folder).
2. In the repo, go to **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to "Deploy from a branch",
   pick your branch, and folder `/ (root)` (or `/docs` if you used that).
4. Save — GitHub will give you a live URL in a minute or two.
